## Deploy options
1. **Netlify** (free)
   - Drag-and-drop this folder into Netlify.
2. **GitHub Pages** (free)
   - Push to GitHub, enable Pages on the repo.
3. **Hosting provider**
   - Upload files via FTP.

## Contact form
The booking form currently uses `mailto:` which opens the user's email client. For a hosted form, use Netlify Forms or a service like Formspree.
